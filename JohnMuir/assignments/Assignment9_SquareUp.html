100 POINTS

Program your robot to drive a precise 4 ft × 4 ft square with turn signals and beeps. Then race it on the course — fastest clean lap wins.

Requirements (must include all 5)

  1. Drive a complete square — 4 straight sides, 4 right turns, return to start.
  2. Right turn signal before every turn — light up the right column of the 5×5 matrix before each turn using light_matrix.set_pixel().
  3. Beep before every turn — audible signal before each turn using sound.beep().
  4. Stay in the lane — robot must stay inside the 1-foot wide taped lane on the course.
  5. Tuned for the course — straight sections should be approximately 4 feet, turns should be approximately 90°.

Starter Code

This drives one side and makes one turn. Extend it to complete the full square.

from hub import light_matrix, sound
import motor_pair
import runloop

motor_pair.pair(motor_pair.PAIR_1, 0, 1)

async def right_blinker():
    # Light up right column of the matrix
    for y in range(5):
        light_matrix.set_pixel(4, y, 100)
    await runloop.sleep_ms(500)
    light_matrix.clear()

async def main():
    light_matrix.write("GO")
    await runloop.sleep_ms(500)

    # --- Side 1 ---
    # Drive forward
    await motor_pair.move_for_time(
        motor_pair.PAIR_1, 2000, 0, velocity=50)

    # Signal and turn
    await right_blinker()
    await sound.beep(800, 300, 100)
    await motor_pair.move_for_time(
        motor_pair.PAIR_1, 800, 100, velocity=30)

    light_matrix.write("DONE")

runloop.run(main())

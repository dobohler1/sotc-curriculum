100 POINTS

Your robot learned to move and make sounds. Now it learns to see. In this assignment, your robot must use the distance sensor to detect an obstacle and react to it — all on its own.

Requirements (must include all 4)

  1. Drive forward continuously — use motor_pair.move() to start driving, not move_for_time(). The robot decides when to stop, not a timer.
  2. Stop when it detects an obstacle — use the distance sensor inside a while True: loop. Stop the motors when something is within 15cm (or your own threshold).
  3. Signal that it stopped — when the robot detects the obstacle, it must do two things: play a sound with sound.beep() AND display something on the light matrix.
  4. Avoid the obstacle — after stopping, the robot must either reverse, turn, or both. It cannot just stop and sit there.

from hub import light_matrix, sound, port
import motor_pair, distance_sensor, runloop

motor_pair.pair(motor_pair.PAIR_1, 0, 1)

async def main():
    light_matrix.write("GO")

    # Start driving forward (continuous)
    motor_pair.move(motor_pair.PAIR_1, 0, velocity=40)

    # Keep checking distance sensor
    while True:
        dist = distance_sensor.distance(port.F)
        if dist != -1 and dist < 150:  # Something within 15cm
            motor_pair.stop(motor_pair.PAIR_1)

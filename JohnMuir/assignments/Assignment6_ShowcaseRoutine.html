Assignment 6: Showcase Routine Challenge

SOTC 2026 — John Muir — Python Robotics

GREEN TABLE — 100 POINTS

Design a fully autonomous robot routine. Your program runs on its own from start to finish — no touching the robot once you press play.

Requirements (must include all 6)

  1. At least 4 different movements — forward, reverse, turn left, turn right. Vary the speed and distance.
  2. At least 2 different sounds — different frequencies, durations, or patterns. A single beep doesn't count twice.
  3. At least 1 light matrix display — show a message or image at some point in the routine.
  4. Use a for loop — somewhere in your routine, repeat something using for i in range().
  5. Use a custom function — write at least one async def my_function(): that you call from inside main(). Name it something meaningful (not function1).
  6. Start and end in the same spot — the robot must return to approximately where it started. This requires planning and calibration.

Example Skeleton

This is a starting point, not a complete solution. You need to add more movements and figure out the return trip yourself.

from hub import light_matrix, sound
import motor_pair
import runloop

async def victory_lap():
    # Your custom function here
    await sound.beep(1000, 200, 100)
    await motor_pair.move_for_time(motor_pair.PAIR_1, 1000, 100, velocity=40)

async def main():
    motor_pair.pair(motor_pair.PAIR_1, 0, 1)

    # Show a starting message
    light_matrix.write("GO")
    await runloop.sleep_ms(500)

    # Movement 1: drive forward
    await motor_pair.move_for_time(motor_pair.PAIR_1, 2000, 0, velocity=50)

    # Movement 2: turn right
    await motor_pair.move_for_time(motor_pair.PAIR_1, 800, 100, velocity=30)

    # Beep pattern using a for loop
    for i in range(3):
        await sound.beep(600, 150, 100)
        await runloop.sleep_ms(200)

    # ... add more movements, sounds, and your return trip ...

    # Call your custom function
    await victory_lap()

    light_matrix.write("DONE")

runloop.run(main())

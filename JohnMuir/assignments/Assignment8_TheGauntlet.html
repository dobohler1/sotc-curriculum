The Course

Coach G sets up a course with 2–3 obstacles (books, boxes, or cones). Your robot starts at one end and must reach the other end by detecting and avoiding obstacles along the way.

  • Start zone: Robot begins here. You press play and step back.
  • Obstacles: 2–3 objects placed in the path. Your robot must not hit them.
  • Finish zone: The end of the course. Getting here = success.

You get 3 attempts. Your best run counts.

Requirements (must include all 5)

  1. Autonomous driving — robot drives forward on its own using motor_pair.move().
  2. Obstacle detection — uses the distance sensor in a while True: loop to detect obstacles.
  3. Obstacle avoidance — when an obstacle is detected, the robot must stop and then navigate around it (reverse, turn, or both).
  4. Status signaling — the robot uses sound AND light matrix to indicate what it's doing (e.g., beep when it detects something, display "GO" at start, "DONE" at finish).
  5. Multiple obstacles — the navigation logic must handle more than one obstacle (i.e., the robot keeps going after avoiding the first one, not just stopping).

Starter Code

This code handles basic detect-and-avoid for one obstacle, then repeats. You need to tune the numbers and may want to add more logic.

from hub import light_matrix, sound, port
import motor_pair, distance_sensor, runloop

motor_pair.pair(motor_pair.PAIR_1, 0, 1)

async def avoid_obstacle():
    """Stop, signal, reverse, and turn to avoid."""
    motor_pair.stop(motor_pair.PAIR_1)
    await sound.beep(800, 300, 100)
    light_matrix.write("!")

    # Reverse for 1 second
    await motor_pair.move_for_time(motor_pair.PAIR_1, 1000, 0, velocity=-40)
    # Turn right for 0.8 seconds
    await motor_pair.move_for_time(motor_pair.PAIR_1, 800, 100, velocity=30)

async def main():
    light_matrix.write("GO")
    await runloop.sleep_ms(500)

    for attempt in range(20):  # try up to 20 check-and-avoid cycles
        # Drive forward
        motor_pair.move(motor_pair.PAIR_1, 0, velocity=40)

        # Check for obstacles
        while True:
            dist = distance_sensor.distance(port.F)
            if dist != -1 and dist < 150:  # within 15cm
                await avoid_obstacle()
                break
            await runloop.sleep_ms(50)

    # Done navigating
    motor_pair.stop(motor_pair.PAIR_1)
    await sound.beep(600, 500, 100)
    light_matrix.write("DONE")

runloop.run(main())
Important: The starter code loops 20 times. That's a lot. Your robot might loop forever if it never gets stuck. You can lower that number, or add a second exit condition (like detecting the finish line color). Tune the numbers based on your testing.

Green Table: Finish Line Detection

Add the color sensor to detect a red finish line. Place a red piece of paper at the end of the course.

import color_sensor

# Add this inside your while True loop, after the distance check:
color = color_sensor.color(port.E)
if color == 9:  # Red
    motor_pair.stop(motor_pair.PAIR_1)
    await sound.beep(440, 500, 100)
    await sound.beep(880, 500, 100)
    light_matrix.write("WIN")
    return  # exit the main function entirely
Color sensor values: 0=Black, 1=Violet, 3=Blue, 4=Cyan, 5=Green, 7=Yellow, 9=Red, 10=White. The sensor must be close to the surface to work — it reads the color directly underneath it.

Bug Hunt — Answers

# Bug 1: "motor pair" → needs underscore
import motor_pair

# Bug 2: missing colon after main()
async def main():

# Bug 3: "awalt" → "await"
    await motor_pair.move_for_time(...)

These are the three most common mistakes in our class. If you caught all 3, you're already debugging like a programmer.

Demo Time — How It Works

  1. Coach G calls your name
  2. Bring your laptop to the demo table
  3. Connect to the robot
  4. Show your code on screen
  5. Run it — we all watch the robot
  6. Answer Coach G's question

~90 seconds per student. Stay working at your seat while you wait.

Coach G Will Ask You One Thing

"Point to the line that controls how fast the robot moves."

or: "What would you change to make it go backwards?"

or: "What does this line do?" (points to a line in your code)

You don't need to have everything memorized. You need to understand your own code.

How You're Scored

A 1 is better than a 0. Showing up and trying counts.

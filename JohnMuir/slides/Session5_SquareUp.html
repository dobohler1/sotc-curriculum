Starter Code — One Side + One Turn

This drives forward and turns once. Your job: make it do all 4 sides.

from hub import light_matrix, sound
import motor_pair
import runloop

motor_pair.pair(motor_pair.PAIR_1, 0, 1)

async def main():
    light_matrix.write("GO")

    # Drive forward (one side of the square)
    await motor_pair.move_for_time(
        motor_pair.PAIR_1, 2000, 0, velocity=50)

    # Beep before turning
    await sound.beep(800, 300, 100)

    # Turn right
    await motor_pair.move_for_time(
        motor_pair.PAIR_1, 800, 100, velocity=30)

    light_matrix.write("DONE")

runloop.run(main())

The Two Numbers You Need to Tune

Straight Line = How Long?

# This controls distance
await motor_pair.move_for_time(
    motor_pair.PAIR_1,
    2000,   # ← milliseconds
    0,
    velocity=50)

Time × Speed = Distance
Too short? Robot doesn't reach 4 ft.
Too long? Robot goes past the turn.

Turn = How Long?

# This controls turn angle
await motor_pair.move_for_time(
    motor_pair.PAIR_1,
    800,    # ← milliseconds
    100,   # ← steering (100=hard right)
    velocity=30)

Too little time? Turn is less than 90°.
Too much? Robot over-rotates.
You need exactly 90°.

Turn Signal — Right Blinker

Before every turn, light up the right side of the matrix. Like a car's turn signal.

Right blinker:

async def right_blinker():
    # Light up right column
    for y in range(5):
        light_matrix.set_pixel(4, y, 100)
    await runloop.sleep_ms(500)
    light_matrix.clear()

Call await right_blinker() before each turn.

One Full Side (With Signals)

This is the pattern for one side. Repeat it 4 times to complete the square.

# --- One side of the square ---

await motor_pair.move_for_time(
    motor_pair.PAIR_1, 2000, 0, velocity=50)

# 3. Beep

# 4. Turn right 90 degrees

Tuning Tips

Problem Fix
Robot doesn't go far enough Increase the time (try 2500 or 3000) or increase velocity
Robot goes too far Decrease the time or decrease velocity
Turn is less than 90° (not square) Increase the turn time (try 1000 or 1200)
Turn is more than 90° Decrease the turn time (try 600)
Robot drifts to one side Your motors may run at slightly different speeds. Try steering 2 or -2 instead of 0
Robot doesn't end where it started Small errors compound over 4 turns. Fine-tune the turn time first — that's the biggest source of drift.

Program BLUE — Drive Forward

Red Table: type this exactly. Every letter, every symbol.

from hub import light_matrix
import motor_pair
import runloop

async def main():
    motor_pair.pair(motor_pair.PAIR_1, 0, 1)
    light_matrix.write("GO")
    await motor_pair.move_for_time(motor_pair.PAIR_1, 2000, 0, velocity=50)
    light_matrix.write("DONE")

runloop.run(main())

Program GOLD — Beep & Spin

Green Table or anyone who finishes BLUE: try this one next.

from hub import light_matrix, sound
import motor_pair
import runloop

async def main():
    motor_pair.pair(motor_pair.PAIR_1, 0, 1)
    await sound.beep(800, 500, 100)
    await motor_pair.move_for_time(motor_pair.PAIR_1, 1500, 100, velocity=30)

runloop.run(main())

Code Not Working? Check These First.

These are the mistakes that block 90% of students.

What You Typed What It Should Be Why
awalt await Spelling — there's no L
light matrix light_matrix Underscore, not a space
motor pair motor_pair Same thing — underscore
async def main() async def main(): Colon at the end
No indent after def     (4 spaces) Everything inside main() must be indented

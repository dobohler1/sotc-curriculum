Meet the Sensors

📏
Distance Sensor
Measures how far away something is. Like sonar — sends out a signal and times the bounce.
distance_sensor.distance(port.F)

Returns distance in mm (-1 = nothing detected)

🌈
Color Sensor
Reads the color of a surface underneath the robot. Points straight down.
color_sensor.color(port.E)

Returns a number (9=Red, 7=Yellow, 5=Green...)

How does it know when to stop?

It reads the distance sensor over and over, hundreds of times per second, until it sees something close.

Program SENSOR — Stop Before You Crash

Type this exactly. New parts are highlighted in the comments.

from hub import light_matrix, sound, port
import motor_pair, distance_sensor, runloop

motor_pair.pair(motor_pair.PAIR_1, 0, 1)

async def main():
    light_matrix.write("GO")

    # Start driving (doesn't stop on its own)
    motor_pair.move(motor_pair.PAIR_1, 0, velocity=40)

    # Keep checking distance
    while True:
        dist = distance_sensor.distance(port.F)
        if dist != -1 and dist < 150:  # within 15cm
            motor_pair.stop(motor_pair.PAIR_1)
            await sound.beep(1000, 300, 100)
            light_matrix.write("STOP")
            break
        await runloop.sleep_ms(50)

runloop.run(main())

What's New Here?

motor_pair.move()

Starts driving and keeps going. Unlike move_for_time(), it doesn't stop on its own. You decide when to stop.

while True:

Runs forever (until you break out). The robot checks the sensor, then checks again, then again...

break

Exits the while loop. When the sensor sees something close, we break out and stop driving.

distance_sensor.distance()

Returns distance in millimeters. Returns -1 if nothing is detected. Check for both.

Line by Line — The Sensor Loop

while True:                                   # loop forever
    dist = distance_sensor.distance(port.F)  # read sensor
    if dist != -1 and dist < 150:           # something close?
        motor_pair.stop(motor_pair.PAIR_1)   # stop motors
        await sound.beep(1000, 300, 100)   # beep!
        light_matrix.write("STOP")          # show STOP
        break                                # exit loop
    await runloop.sleep_ms(50)               # wait 50ms

Think of it like this: Drive. Check. Drive. Check. Drive. Check. STOP.

Bonus: The Color Sensor

Your robot can also see colors on the surface below it.

from hub import light_matrix, port
import color_sensor

# Read the color once
color = color_sensor.color(port.E)

if color == 9:     # Red
    light_matrix.write("RED")
elif color == 5:   # Green
    light_matrix.write("GRN")
elif color == 3:   # Blue
    light_matrix.write("BLU")
else:
    light_matrix.write("???")

Color values: 0=Black, 1=Violet, 3=Blue, 4=Cyan, 5=Green, 7=Yellow, 9=Red, 10=White

Sensor Not Working? Check These.

Problem Fix
Distance always shows -1 Check the port letter. If sensor is in port F, use port.F. Look at the cable.
Robot never stops Put something big and flat in front of the sensor (a book). Small or angled objects can be missed.
"distance_sensor not defined" Add import distance_sensor at the top of your code.
"port not defined" Add from hub import port to your import line.
Robot drives but code freezes Make sure you have await runloop.sleep_ms(50) inside your while loop.

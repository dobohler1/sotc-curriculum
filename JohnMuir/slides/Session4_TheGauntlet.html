The Rules

  1. Robot starts at the START zone. You press play. Hands off.
  2. Robot must drive forward and detect obstacles using the distance sensor.
  3. When it detects an obstacle, it must stop, signal (sound or light), and navigate around it.
  4. Robot must reach the FINISH zone.
  5. If the robot hits an obstacle, you can restart — but it costs you points.

You get 3 attempts. Best run counts.

New Concept: Sensor-Based Decisions

Combine if/elif/else with sensor data to make your robot smart.

# Check distance and decide what to do
dist = distance_sensor.distance(port.F)

if dist == -1:
    # Nothing detected — keep driving
    motor_pair.move(motor_pair.PAIR_1, 0, velocity=40)

elif dist < 100:
    # Very close! Stop and turn
    motor_pair.stop(motor_pair.PAIR_1)
    await sound.beep(1000, 200, 100)
    await motor_pair.move_for_time(
        motor_pair.PAIR_1, 800, 100, velocity=30)

elif dist < 250:
    # Getting close — slow down
    motor_pair.move(motor_pair.PAIR_1, 0, velocity=20)

Gauntlet Starter Code

from hub import light_matrix, sound, port
import motor_pair, distance_sensor, runloop

motor_pair.pair(motor_pair.PAIR_1, 0, 1)

async def avoid_obstacle():
    # Stop, reverse, turn
    motor_pair.stop(motor_pair.PAIR_1)
    await sound.beep(800, 300, 100)
    light_matrix.write("!")
    await motor_pair.move_for_time(motor_pair.PAIR_1, 1000, 0, velocity=-40)
    await motor_pair.move_for_time(motor_pair.PAIR_1, 800, 100, velocity=30)

async def main():
    light_matrix.write("GO")
    for attempt in range(20):  # try up to 20 times
        motor_pair.move(motor_pair.PAIR_1, 0, velocity=40)
        while True:
            dist = distance_sensor.distance(port.F)
            if dist != -1 and dist < 150:
                await avoid_obstacle()
                break
            await runloop.sleep_ms(50)
    motor_pair.stop(motor_pair.PAIR_1)
    light_matrix.write("DONE")

runloop.run(main())

Green Table Stretch: Finish Line Detection

Add the color sensor to detect a red piece of paper as the finish line.

import color_sensor

# Inside your while True loop, add this check:
color = color_sensor.color(port.E)
if color == 9:  # Red detected!
    motor_pair.stop(motor_pair.PAIR_1)
    await sound.beep(440, 500, 100)
    await sound.beep(880, 500, 100)
    light_matrix.write("WIN")
    break  # exit the whole navigation loop

Place a red piece of paper at the finish zone. When the color sensor passes over it, the robot knows it's done.

Tips for the Course Run

Situation What to Do
Robot drives into the obstacle Lower the distance threshold (try 200 instead of 150) or slow down the robot.
Robot turns but then drives into the same obstacle Increase the turn time (try 1200ms instead of 800ms) or turn at a wider angle.
Robot keeps turning in circles Alternate turn direction. Or reverse more before turning.
Robot never detects the obstacle Make sure the obstacle is big and flat (a book, not a pencil). Check sensor port.
Code runs but nothing happens Check motor port numbers. Check that motor_pair.pair() is called before main().

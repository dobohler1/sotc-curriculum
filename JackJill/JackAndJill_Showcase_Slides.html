Step 3: Add the Backup Beep

Real trucks beep when they reverse. So should your robot.

⚑ when program starts
⚙ set movement motors to A + B
⚙ move forward for 2 seconds
⏳ wait 1 seconds
♫ play beep 72 for 0.5 seconds

Click Play. Did it beep before reversing?

The Same Thing in Python

from hub import light_matrix, sound
import motor_pair, runloop

async def main():
    motor_pair.pair(motor_pair.PAIR_1, 0, 1)
    light_matrix.show_image(
        light_matrix.IMAGE_HAPPY)
    await motor_pair.move_for_time(
        motor_pair.PAIR_1,
        2000, 0, velocity=50)
    await runloop.sleep_ms(1000)
    light_matrix.show_image(
        light_matrix.IMAGE_SAD)
    await sound.beep(800, 500, 100)
    await motor_pair.move_for_time(
        motor_pair.PAIR_1,
        2000, 0, velocity=-50)

runloop.run(main())

Same robot. Same behavior. Python is the language used by engineers at Google, NASA, and Tesla.

THE RACE

The 10-Foot Round Trip Challenge

1. Your robot drives forward exactly 10 feet

2. It stops

3. It beeps

4. It reverses back to the exact starting position

You get 3 attempts. Best run counts.

Tuning Your Robot

To go exactly 10 feet you need to experiment with these two numbers:

⚙ move forward for ??? seconds
Duration (seconds) How long it drives. Try 4, 6, 8
Speed Add a set movement speed block. Try 30, 50, 75

Hint: slow and long is easier to control than fast and short.

Forward and backward duration should match!

Test, adjust, repeat. That's how real engineers do it.

Race Scoring

Robot reaches the 10-foot line 5 pts
Robot returns within 12 inches of start 5 pts
Robot returns within 6 inches of start 10 pts
Robot beeps before reversing 5 pts
Light display changes (forward vs. reverse) 5 pts
Maximum score 30 pts
